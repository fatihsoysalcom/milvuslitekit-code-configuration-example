# MilvusLiteKit Code Configuration Example

This Python script demonstrates how to configure MilvusLite collections, schemas, and indexes entirely through Python code. It connects to an in-memory Milvus instance, defines a schema with a primary key and a vector field, inserts sample data, creates an index, and performs a basic vector search.

## Language

`python`

## How to Run

1. Ensure you have pymilvus installed (`pip install pymilvus`).
2. Run the script using `python milvus_config_example.py`.
